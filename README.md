# odin-recipes


A basic HTML recipe website created as part of The Odin Project curriculum.

## Project Description

This project is a simple multi-page website built using only HTML.  
The main page contains links to several recipe pages. Each recipe page includes an image, description, ingredients list, and step-by-step instructions.

The goal of this project is to practice core HTML structure and page linking.

## Skills Demonstrated

- HTML boilerplate structure
- Headings and paragraphs
- Links and relative paths
- Images with alt text
- Unordered and ordered lists
- Multi-page website structure
- Basic Git and GitHub workflow (stage, commit, push)
